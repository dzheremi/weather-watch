    class StationsController

    API controller for access an index of stations.

    Public Instance Methods

    show() click to toggle source

    Methods returns a JSON file based on the StationFeed class. Can be called with one of the following URLs:

    • /stations - Returns all stations for all states.

    • /stations/{{state_abbreviation}} - Returns all stations for a given state (using its abbreviation).

    # File app/controllers/stations_controller.rb, line 9
def show

  if params[:state].nil?
    states = State.all

  else
    states = State.where(abbreviation: params[:state].downcase)

  end

  feed = StationFeed.new
  feed.set_stations(states)
  render json: feed

end
